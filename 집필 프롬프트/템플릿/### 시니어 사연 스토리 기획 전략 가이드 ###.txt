**추천 제목 예시:**

- "손주 용돈 3천만 원 줬더니 연락이 끊겼습니다"

## 🎯 시니어 막장 공식 (검증된 패턴)

### 황금 공식: **일상 → 의심 → 충격 → 배신 → 반전 → 복수 → 카타르시스**

```
1. 평범한 일상 (공감대 형성)
   ↓
2. 작은 이상 징후 (의심 시작)
   ↓
3. 충격적 발견 (감정 고조)
   ↓
4. 가족의 배신 확인 (절망)
   ↓
5. 예상 못한 반전 (재미 극대화)
   ↓
6. 주인공의 반격 (통쾌함)
   ↓
7. 정의 구현 또는 해피엔딩 (만족)
```

## 🔥 클릭률 폭발 제목 공식

### 패턴 1: **[일상 + 배신]**
```
"[평범한 관계]가 [충격적 행동]을 했습니다"

예시:
- "며느리가 제 약에 독을 탔습니다"
- "아들이 제 집을 몰래 팔았어요"
- "손주가 제 카드로 5천만 원을 썼습니다"
```

### 패턴 2: **[시간 + 반전]**
```
"[오랜 시간]... 그 순간 [충격 진실]"

예시:
- "30년 동거... 알고 보니 법적 배우자가 아니었어요"
- "50년 숨긴 비밀... 제 자식이 아니었습니다"
- "20년 간병... 남편은 재산을 다 빼돌렸더군요"
```

### 패턴 3: **[행동 + 후회]**
```
"[과거 행동]했더니 [현재 결과]..."

예시:
- "재산 다 물려줬더니 요양원 버려졌어요"
- "집 팔아 아들 사업 도왔더니 연락 두절됐습니다"
- "손주 뒷바라지 10년... 친손주가 아니었어요"
```

### 패턴 4: **[발견 + 충격]** ⭐ 가장 강력
```
"[일상 물건/장소]에서 [충격 진실] 발견했어요"

예시:
- "며느리 서랍에서 제 통장이 나왔습니다"
- "아들 차 트렁크에서... 절 요양원 보낼 서류가"
- "남편 휴대폰에서 20년 숨겨온 비밀을 봤어요"
```

## 💡 시나리오 작가의 핵심 조언

### ✅ 반드시 포함할 요소

1. **명확한 악역**
   - 며느리 > 아들 > 사위 > 손주 순으로 효과적
   - 악역은 철저히 나쁘게 (중간은 안 됨)

2. **돈/재산 구체적 금액**
   - "많은 돈" (X)
   - "3억 원" (O)
   - 숫자가 있어야 현실감

3. **일상 소재로 시작**
   - 액자, 통장, 서랍, 휴대폰
   - "나도 저거 있는데" 공감

4. **2단 반전 이상**
   - 1차: 며느리가 재산 노림
   - 2차: 알고 보니 전문 사기꾼
   - 3차: 피해자가 나만이 아님

5. **정의 구현 필수**
   - 법적 처벌
   - 사회적 망신
   - 악역의 몰락
   - (단, 너무 잔인하면 안 됨)

---

### ❌ 피해야 할 요소

1. **애매한 결말**
   - 시니어는 명확한 결론 선호
   - "생각해 보세요" 스타일 (X)

2. **너무 복잡한 플롯**
   - 등장인물 5명 이하
   - 시간선 2개 이하
   - 단순명쾌하게

3. **불행한 엔딩**
   - 주인공이 지는 결말 (X)
   - 악역이 이기는 결말 (X)
   - 양비론 (X)

4. **성적/폭력적 표현**
   - 직접적 묘사 금지
   - 암시 정도만
   - 가족 시청 가능 수준

5. **현대 용어 남발**
   - "가스라이팅" (X) → "정신적 학대" (O)
   - "마인드 컨트롤" (X) → "세뇌" (O)

   - 반전 + 정의

9. **"재혼한 남편, 알고 보니 제 재산만 노린 전문 사기꾼이었어요"**
   - 사랑 + 배신 + 반전

10. **"70세에 시작한 유튜브... 절 버린 자식들이 찾아왔습니다"**
    - 현대적 + 복수 + 희망

## 📈 조회수 예측 공식

```
높은 조회수 = 
(공감도 × 0.3) + 
(충격도 × 0.3) + 
(반전 × 0.2) + 
(카타르시스 × 0.2)

※ 각 항목 10점 만점
```

**예시 채점:**
"며느리가 제 사진을 액자에서 빼놨습니다"
- 공감도: 9점 (누구나 겪을 수 있음)
- 충격도: 8점 (사진 빼기 → 재산 노림)
- 반전: 10점 (전문 사기꾼)
- 카타르시스: 9점 (법적 처벌)
- **총점: 8.9/10** ✅ 고조회수 예상

## 💬 작가로서의 최종 조언

시니어 사연 스토리의 핵심은 **"공감 → 분노 → 통쾌함"**입니다.

1. **첫 3분**: 일상 공감 (나도 저런 적 있는데...)
2. **중간**: 배신으로 분노 (저런 나쁜 놈들!)
3. **후반**: 반전과 복수 (속이 다 시원하다!)
4. **결말**: 교훈과 희망 (나도 용기 내야지)

**가장 중요한 것:**
막장 요소는 있되, **과하지 않게**.
자극적이되, **품위는 유지**.
현실적이되, **희망은 남기기**.

이 균형을 맞추면 시니어들이 **보고 또 보는** 콘텐츠가 됩니다! 🎬✨
