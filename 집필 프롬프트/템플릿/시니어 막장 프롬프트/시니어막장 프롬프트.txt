작성된 등장인물과 플롯을 기반으로 시나리오를 작성할거야. 
나는 한 씬씩 집필하는걸 선호하기 때문에 내가 한 씬씩 집필을 부탁하면 그 때 집필해줘.
단, 집필 지침이 있기 때문에 아래 지침에 충실해서 집필하도록 해. 
아래 지침을 모두 확인했으면 바로 집필을 시작하지 말고, 이해했다고 대답해
1. 분량과 제목
각 씬은 한글 기준으로 2000자에 맞춰 집필한다.
씬 제목이 없으면 소제목을 직접 정해 기록한다. 형식은 다음과 같다. 씬 1: 제목
2. 대사와 나레이션 구성
전체 분량에서 대사는 20% 비중으로 하고 나레이션은 80% 비중으로 구성한다. 
나레이션은 극의 분위기와 등장인물의 심리와 행동을 묘사한다.
플롯에 있는 대사는 그대로 옮기지 않고 상황에 맞게 변형해 쓴다.
3. 나레이션 원칙
나레이션은 3인칭 관찰자의 시점으로 진행한다.
나레이션은 존댓말로 서술한다. - 문장 끝 어미는 -요, -습니다, -죠가 적절히 번갈아 나오게 할 것.
나레이션은 문장 단위로 줄바꿈한다. 
문장이 끝나지 않으면 줄바꿈하지 않는다.
4. 심리와 행동 묘사
인물의 심리나 행동을 생생하게 묘사해 독자가 몰입할 수 있도록 만든다.
5. 씬 전개와 연결
씬 간 중복과 반복을 금지한다. 앞에서 이미 언급한 문장을 다시 쓰지 않는다.
6. 문체와 문장 구성
70대 이상 시니어 독자가 쉽게 몰입할 수 있도록 쉬운 단어와 문장을 사용한다.
7. 집필 방식
씬 집필을 마친 뒤에는 추가 설명을 붙이지 않는다. 예를 들어 씬 6을 집필했습니다 같은 첨언은 하지 않는다.
8. 진행 지시
씬 집필이 끝나고 내가 "ㅇㅇ"를 입력하면 다음 씬을 집필한다.



## 🎯 시니어 막장 공식 (검증된 패턴)

### 황금 공식: **일상 → 의심 → 충격 → 배신 → 반전 → 복수 → 카타르시스**

```
1. 평범한 일상 (공감대 형성)
   ↓
2. 작은 이상 징후 (의심 시작)
   ↓
3. 충격적 발견 (감정 고조)
   ↓
4. 가족의 배신 확인 (절망)
   ↓
5. 예상 못한 반전 (재미 극대화)
   ↓
6. 주인공의 반격 (통쾌함)
   ↓
7. 정의 구현 또는 해피엔딩 (만족)
```
## 핵심 구조 패턴

### 1. **기본 공식: [상황 설정] + [결정적 순간/발견] + [극적 행동/결과]**

**예시:**
- "며느리와 일본여행 가는데" (상황) + "뒷자리 승객의 쪽지를 본 순간" (발견) + "경악하고 도망쳤습니다" (행동)
- "아들집 김치냉장고 열었다가" (상황) + "안에 있던 것들" (발견) + "아들은 도망치는데" (결과)

### 2. **3단계 긴장 고조 구조**

**1단계: 평범한 일상**
- "며느리와 커피 마시는데"
- "병원에서 진료 기다리던 중"
- "택시타고 며느리와 여행가는데"

**2단계: 결정적 힌트/경고**
- "기사가 몰래 다가와서 '어르신, 지금 당장 도망가세요'"
- "간호사가 슬쩍 다가와서 '어르신, 며느리 조심하세요'"
- "카페 직원이 다가와서 '어르신, 며느리가...'"

**3단계: 충격적 반응/복수**
- "목숨 걸고 뛰었습니다"
- "온몸이 굳어버렸습니다"
- "아들집 통째로 날려버린 할아버지"

### 3. **주요 시나리오 유형**

**A. 제3자 경고형** (조회수 높음 ⭐)
```
[가족과 함께 있는 상황] + [제3자의 경고 메시지] + [도망/신고]
```
- 택시기사, 카페 직원, 간호사, 뻥튀기 아저씨 등이 경고
- "어르신, 조심하세요" / "빨리 도망가세요"
- 긴장감과 호기심 극대화

**B. 우연한 발견형**
```
[일상적 행동] + [충격적 발견] + [경찰 신고/복수]
```
- "김치냉장고 열었다가"
- "유품 정리하는데"
- "손주의 잠꼀대 한마디에"

**C. 모욕/무시 → 통쾌한 복수형**
```
[며느리/자식의 무례한 행동] + [참다 참다] + [재산 정리/집 팔기]
```
- "문 앞에서 신발도 못 벗게 하던 날"
- "화장실 출입금지 쪽지 붙이던 날"
- "밥상을 치워버리던 날"

**D. 버림 → 대역전형**
```
[자식에게 버림받음] + [어려운 시기] + [인생 대역전/복수]
```
- "공항에 버려진 → 복수"
- "노숙생활 → 귀인 만남 → 인생 대역전"

### 4. **제목 구성 요소**

**감정 증폭 키워드:**
- 경악, 기겁, 굳어버렸습니다, 주저앉았습니다
- 피눈물 나게, 처절한, 통째로

**긴급성/긴장감 표현:**
- "지금 당장", "빨리", "목숨 걸고"
- "뒤도 안 돌아보고", "버선발로", "맨발로"

**극단적 행동:**
- "싹다 뒤집어버렸습니다"
- "감옥 보냈습니다"
- "팔아버린 할아버지"
- "날려버린"

### 5. **시간대별 긴장 장치**

- "새벽 2시에"
- "밤중에"
- "아내 기일에"
- "장례식 끝나고"

### 6. **효과적인 구두점 사용**

- **마침표 대신 말줄임표**: "며느리가..." (궁금증 유발)
- **느낌표로 강조**: "어르신! 빨리 도망가세요!"
- **인용부호로 생생함**: "엄마, 절대 오지 말아요"

## 성공 공식

**가장 높은 조회수 패턴:**
1. 제3자가 등장해서 경고하는 구조
2. 일상적 상황에서 우연히 발견하는 구조  
3. 구체적인 모욕 → 통쾌한 복수 구조
4. "~하던 날" 시간 표현으로 결정적 순간 강조

**핵심은 '안전해 보이는 상황 → 숨겨진 위험 발견 → 극적 행동'의 3단 구조입니다.**
